Este programa, escrito em linguagem C, foi desenvolvido como parte de um trabalho acadêmico com o objetivo de organizar tarefas. Além das funcionalidades básicas de criação, listagem e exclusão de tarefas, o programa permite carregar tarefas previamente salvas em um arquivo binário.

Funcionalidades do Programa:

Criar Tarefa: Os usuários podem adicionar novas tarefas à lista, fornecendo um título e uma descrição para cada uma.

Listar Tarefas: O programa exibe todas as tarefas presentes na lista, mostrando seus títulos e descrições.

Deletar Tarefa: Os usuários têm a opção de excluir tarefas específicas da lista, removendo-as permanentemente.

Carregar Tarefas Salvas: O programa é capaz de carregar tarefas previamente salvas em um arquivo binário, permitindo a continuidade do trabalho anterior.

Salvar em Arquivo Binário: As tarefas são armazenadas em um arquivo binário para garantir a persistência dos dados entre diferentes execuções do programa.

Implementação:

O programa foi escrito em linguagem C.

Utilizou-se operações de entrada e saída de arquivos para carregar e salvar tarefas em formato binário.

Estruturas de dados como arrays foram empregados para armazenar e manipular as tarefas dentro do programa.

Foi implementada uma função para carregar as tarefas salvas no início da execução do programa, se houver.
